力扣1446.连续字符 Easy#

题目

给你一个字符串 s ，字符串的「能量」定义为：只包含一种字符的最长非空子字符串的长度。

请你返回字符串的能量。

code

class Solution:
    def maxPower(self, s: str) -> int:
        if len(s) == 1:
            return 1
        elif len(s) == 0:
            return 0
        result = 1
        arr = []
        for i in range(1,len(s)):
            if s[i] == s[i-1]:
                result += 1
            else:
                arr.append(result)
                result  = 1
        arr.append(result)
        return max(arr)

题解

常规第一题的难度，统计每一个连续字符的数量，然后return最大值就可以了

力扣1447.最简分数 Medium#

题目

给你一个整数 n ，请你返回所有 0 到 1 之间（不包括 0 和 1）满足分母小于等于 n 的 最简分数 。分数可以以 任意 顺序返回。

code

class Solution:
    def simplifiedFractions(self, n: int) -> List    [str]:
        arr = []
        for i in range(1, n):
            for j in range(i + 1, n + 1):
                if math.gcd(i,j) == 1:
                    arr.append("{}/{}".format(i,j))
        return arr

题解

本体的核心是解决最简分数的判断方法，最简分数就是分子和分母不能再进行约分，即分子分母的最大公约数是1，math.gcd(分子，分母)==1，即说明是最简分数。

1450. 在既定时间做作业的学生人数 Easy

题目

给你两个整数数组 startTime（开始时间）和 endTime（结束时间），并指定一个整数 queryTime 作为查询时间。

已知，第 i 名学生在 startTime[i] 时开始写作业并于 endTime[i] 时完成作业。

请返回在查询时间 queryTime 时正在做作业的学生人数。形式上，返回能够使 queryTime 处于区间 [startTime[i], endTime[i]]（含）的学生人数。

code

class Solution:
    def busyStudent(self, startTime: List[int], endTime: List[int], queryTime: int) -> int:
        result = 0
        for i,j in zip(startTime,endTime):
            if i<=queryTime<=j:
                result += 1
        return result

题解

常规第一题难度，没什么好说的

1451. 重新排列句子中的单词 Medium

题目

「句子」是一个用空格分隔单词的字符串。给你一个满足下述格式的句子 text :

句子的首字母大写
text 中的每个单词都用单个空格分隔。

请你重新排列 text 中的单词，使所有单词按其长度的升序排列。如果两个单词的长度相同，则保留其在原句子中的相对顺序。

请同样按上述格式返回新的句子。

code

class Solution:
    def arrangeWords(self, text: str) -> str:
        result = text.split()
        result = sorted(result,key = lambda x:(len(x)))
        return " ".join(result).capitalize()

题解

这里就是python的优势，可以直接使用sorted()进行排序，key = lambda 可以对排序规则进行自定义，这道题我学习到了，如果我没有第二个定义规则的话，在len(x)相同的情况下，会按照原句中的相对顺序进行排序。这道题如果改变为如果两个单词的长度相等，按字典序排序，那么key = lambda 在定义len(x)后还要在加入x[0]。

1452. 收藏清单 Medium

题目

给你一个数组 favoriteCompanies ，其中 favoriteCompanies[i] 是第 i 名用户收藏的公司清单（下标从 0 开始）。

请找出不是其他任何人收藏的公司清单的子集的收藏清单，并返回该清单下标。下标需要按升序排列。

code

class Solution:
    def peopleIndexes(self, favoriteCompanies: List[List[str]]) -> List[int]:
        result = []
        for i in range(len(favoriteCompanies)):
            flag = 1
            for k in range(len(favoriteCompanies)):
                if i == k:
                    pass
                else:
                    if set(favoriteCompanies[i]) <= set(favoriteCompanies[k]):
                        flag = 0
            if flag == 1:
                result.append(i)
        return result

题解

这道题也可以充分体现py的优越性，在判断是否子集上，可以直接set<=表示子集。
这样暴力就可以解决了

1453. 圆形靶内的最大飞镖数量 Hard

题目

墙壁上挂着一个圆形的飞镖靶。现在请你蒙着眼睛向靶上投掷飞镖。

投掷到墙上的飞镖用二维平面上的点坐标数组表示。飞镖靶的半径为 r 。

请返回能够落在 任意 半径为 r 的圆形靶内或靶上的最大飞镖数。

思考

这次Hard留给我充分的思考时间，我看群友想到了三个点确定一个圆心，然后遍历所有圆心看覆盖的最多点是哪个，我在执行过程中遇到一个小问题，过了所有的实例，先贴下三个点确定圆心的代码，以后有时间写出来再贴完整代码。

x1, y1 = points[i][0], points[i][1]
x2, y2 = points[j][0], points[j][1]
x3, y3 = points[k][0], points[k][1]
a = ((y2 - y1) * (y3 * y3 - y1 * y1 + x3 * x3 - x1 * x1) - (y3 - y1) * (y2 * y2 - y1 * y1 + x2 * x2 - x1 * x1)) / (2.0 * ((x3 - x1) * (y2 - y1) - (x2 - x1) * (y3 - y1)))
b = ((x2 - x1) * (x3 * x3 - x1 * x1 + y3 * y3 - y1 * y1) - (x3 - x1) * (x2 * x2 - x1 * x1 + y2 * y2 - y1 * y1)) / (2.0 * ((y3 - y1) * (x2 - x1) - (y2 - y1) * (x3 - x1)))
