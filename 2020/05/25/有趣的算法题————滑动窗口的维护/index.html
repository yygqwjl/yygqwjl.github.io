有趣的算法题————滑动窗口的维护
发表于 2020-05-25| 更新于 2020-06-01|算法

此题来自力扣第190场周赛T2

链接：https://leetcode-cn.com/problems/maximum-number-of-vowels-in-a-substring-of-given-length

题目

给你字符串 s 和整数 k 。

请返回字符串 s 中长度为 k 的单个子字符串中可能包含的最大元音字母数。

英文中的 元音字母 为（a, e, i, o, u）。

code

class Solution:
    def maxVowels(self, s: str, k: int) -> int:
        left,right = 0,k-1
        now = 0
        for i in range(k):
            if s[i] in ["a","e","i","o","u"]:
                now += 1
        result = []
        result.append(now)
        while right < len(s)-1:
            if s[left] in ["a","e","i","o","u"]:
                now -= 1
            if s[right+1] in ["a","e","i","o","u"]:
                now += 1
            left += 1
            right += 1
            result.append(now)
        return max(result)

题解

本题的核心就是滑动窗口，但是需要对窗口进行维护，滑动窗口的模板在逆向思维的那篇文章写过了，这篇文章的目的主要是理解题意，对窗口进行维护，从而降低时间复杂度。

贴一下第一次暴力超时的代码

class Solution:
    def maxVowels(self, s: str, k: int) -> int:
        left,right = 0,k-1
        result = [0]
        def main(s):
            cnt = 0
            for i in s:
                if i in ["a", "e", "i", "o", "u"]:
                    cnt += 1
            return cnt
        while right <= len(s)-1:
            result.append(main(s[left:right+1]))
            left += 1
            right += 1
        return max(result)

思考

这种做法就是纯粹暴力每一个窗口，定义了一个main()函数来遍历每个窗口找到最大值，而这篇文章就是要将优化暴力窗口，在窗口的滑动中，只要求出第一个窗口的元音字母数之后优化这个值就可以了(代码中的now),如果窗口的第一个值是元音，在下一步滑动中，它将离开窗口，所以now-1，如果窗口之外的第一个值是原因，在下一步滑动中，它将进入窗口，所以now+1。这样复杂度就降低了很多，在看到题之后认真读题，这种题第一眼就知道是滑动窗口，但是要思考的就是如何进行维护，优化窗口的复杂的。
