易错易混知识点
发表于 2020-05-01| 更新于 2020-05-26|总结基础知识

判断题

python是解释型的程序设计语言。 (√)

Python的浮点数常量也有十进制、二进制、八进制和十六进制等表示方式。(×)

Python可以使用For作为变量名。(√)

Python函数支持可变数量的参数，实参用“ *参数名 ”表示(×)

python是脚本语言，主要用作系统编程和Web访问的开发语言(×)

python字符编码使用ASCII编码(×)

python解释器把python代码一次性翻译成目标代码，然后执行(×)

填空

除了4个空格和8个空格，Python还经常用(tab)符号来表示代码的缩进。

在Windows操作系统中选下载Python解释器的时候需要注意操作系统的版本，X86表示(32位的系统)

正则表达式元字符(x)用来表示该符号前面的字符或子模式1次或多次出现

print(0.1 + 0.2 == 0.3)的输出结果是(False)

python为源文件指定的默认字符编码是 (UTF-8)

使用pip工具查看当前已安装的Python扩展库的完整命令是(pip list）

Python语句既可以采用交互式的命令执行方式，又可以采用程序执行方式。

在Python集成开发环境中，可使用F5运行程序。

小知识

对于一些非布尔类型的数值在需要使用布尔值的时候，值为0的数字以及空集（字符串、列表、元组、集合等）都将被理解为False

python字符编码可以使用ASCII编码和Unicode编码

高级语言根据计算机执行机制的不同可分为两类，静态语言和脚本语言。静态语言采用编译方式执行，脚本语言采用解释方式执行

IPO模式包括Input，Process，Output

解释是将源代码逐条转换成目标代码同时逐条运行目标代码的过程。执行解释的计算机程序为解释器

sum不是保留字

适合编写系统软件不是python的语言特点

await是保留字

保留字注意大小写

for…in…不能遍历浮点数，因为浮点数的个数是无限的

异常和错误是不同的概念，异常仅指程序运行层面的错误，而错误范围更广泛，还包括程序的逻辑错误等。

for是遍历循环，while是无限循环

二分支是if-else，多分支是if-elif-else

如果一个函数需要以多种形式来接受实参，定义时一般把位置参数放在最前面，然后是默认参数，接下来是一个星号的可变长度参数，最后是两个星号的可变长度参数。

在题干中看清执行完程序还是执行完函数！
全局变量和局部变量可以重名。

表达键值对的字典类型最可能属于的数据组织维度是高维数据

python程序拓展名py，GUI程序是pyw，源文件程序编译后的文件拓展名是pyc

在位运算中，每左移一位相当于乘以2，每右移一位相当于除以2。

在Python语句行中使用多条语句，语句之间使用分号分割。

有关re的函数中flags，re.I 表示忽略大小写。

默认splitelines参数keepends为False，意思是不保留每行结尾的\n, 而keepends为True时，分割的每一行里尾部会有\n。
总结，splitlines是按行分割字符串，返回值也是个列表。

表达式 1<2 == 2 的值为2

表达式 3+2 << 1 的值为10(先进行加减操作，再进行位运算)

5>3 is False 的值为 False

(5>3) is True 的值为 True

eval(“3*2”+”22”)的值为666

3 and 5 的值为 5
