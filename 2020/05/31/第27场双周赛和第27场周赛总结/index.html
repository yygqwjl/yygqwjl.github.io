双周赛

leetcode 5408.通过翻转子数组使两个数组相等 Easy

题目

给你两个长度相同的整数数组 target 和 arr 。

每一步中，你可以选择 arr 的任意 非空子数组 并将它翻转。你可以执行此过程任意次。

如果你能让 arr 变得与 target 相同，返回 True；否则，返回 False 。

code

class Solution:
    def canBeEqual(self, target: List[int], arr: List[int]) -> bool:
        return sorted(target) == sorted(arr)

题解

这道题就是使我全网第一个完成的题，最近的列表处理做的很多，因为我读题比较快，所以立马理解题意就是判断两个数组是否相等，所以利用sorted函数立刻ac

leetcode 1461. 检查一个字符串是否包含所有长度为 K 的二进制子串 Medium

题目

给你一个二进制字符串 s 和一个整数 k 。
如果所有长度为 k 的二进制字符串都是 s 的子串，请返回 True ，否则请返回 False 。

code

class Solution:
    def hasAllCodes(self, s: str, k: int) -> bool:
        import itertools
        p = ["01"]*k
        x = itertools.product(*p)
        for i in x:
            if "".join(i) not in s:
                return False
        return True

题解

itertools和collections是我最喜欢的两个库，一个可以帮助我解决基础类的排列组合的问题，另一个帮助我解决字符统计的问题。

这道题我的思路是构造所有长度为k的二进制字串，然后用in遍历字符，如果一个为False，可以判定False。
构造使用了itertools，从“01”中挑选K次，就是长度为K的二进制字串。先构造p，然后在itertools.product()中解包。

我看了几个题解，也都是暴力，不过思路是在字符串遍历长度的K的字符，最后看不同的字符是否有2**K次个。

周赛

leetcode 5424. 数组中两元素的最大乘积 Easy

题目

给你一个整数数组 nums，请你选择数组的两个不同下标 i 和 j，使 (nums[i]-1)*(nums[j]-1) 取得最大值。

请你计算并返回该式的最大值。

code

class Solution:
    def maxProduct(self, nums: List[int]) -> int:
        k = [i-1 for i in nums]
        k = sorted(k)
        return max(k[0]*k[1],k[-1]*k[-2])

题解

因为之前做过数组中三个元素相乘的最大值，这道题算是前面那道题的简单版，先说一下三个元素相乘的最大值，先排序，然后因为元素可能有正有负，所以返回的是1.最大的三个正数的乘积2.前两个大的正数与最小的负数的乘积3.最小的两个负数与最大的正数的乘积。
代码：

nums.sort()
return max(nums[-1]*nums[-2]*nums[-3],nums[-1]*nums[-2]*nums[0],nums[0]*nums[1]*nums[-1])

而这道题只有两个数，先map一下把数列中的数字全部减1，然后直接返回两个最大正数乘积和两个最小负数的乘积中的最大值就可以了

leetcode 5425. 切割后面积最大的蛋糕 Medium##

题目

矩形蛋糕的高度为 h 且宽度为 w，给你两个整数数组 horizontalCuts 和 verticalCuts，其中 horizontalCuts[i] 是从矩形蛋糕顶部到第 i 个水平切口的距离，类似地， verticalCuts[j] 是从矩形蛋糕的左侧到第 j 个竖直切口的距离。

请你按数组 horizontalCuts 和 verticalCuts 中提供的水平和竖直位置切割后，请你找出 面积最大 的那份蛋糕，并返回其 面积 。由于答案可能是一个很大的数字，因此需要将结果对 10^9 + 7 取余后返回。

code

class Solution:
    def maxArea(self, h: int, w: int, horizontalCuts: List[int], verticalCuts: List    [int]) -> int:
        horizontalCuts.append(0)
        horizontalCuts.append(h)
        verticalCuts.append(0)
        verticalCuts.append(w)
        horizontalCuts.sort()
        verticalCuts.sort()
        listh,listv = [],[]
        for i in range(1,len(horizontalCuts)):
            listh.append(horizontalCuts[i]-horizontalCuts[i-1])
        for j in range(1,len(verticalCuts)):
            listv.append(verticalCuts[j]-verticalCuts[j-1])
        return (max(listh)*max(listv))%(10**9+7)

题解

这道题很长还有也很难理解，需要的是阅读题意理解的能力，其实就是按照horizontalCuts和verticalCuts切完后，找到最大的蛋糕，不要忘记还有两个边界，找到最长的长和最长的宽相乘就可以了。

最重要的是要记得取余，因为没有取余wa了一次，以后要长点记性。

ps：变量名太长也是这道题不好做的原因
