井字棋大作业

code

def game_mune():
    print("       Tic-tac-toe      ")
    m = input("please enter any key to start the game!")

def display_borad():
    print("-----------")
    print("|{} | {} | {}|".format(borad[0][0],borad[0][1],borad[0][2]))
    print("-----------")
    print("|{} | {} | {}|".format(borad[1][0],borad[1][1],borad[1][2]))
    print("-----------")
    print("|{} | {} | {}|".format(borad[2][0],borad[2][1],borad[2][2]))
    print("-----------")

def full_borad(borad):
    flag = True
    for i in borad:
        if 0 in i:
            flag = False
    if flag == True:
        print("tie")
        return False
    else:
        return True

def player(count):
    if count % 2 == 0:
        print("now player is 1")
        a = int(input("please input row:"))
        if a > 3 or a < 1:
            a = int(input("Out of range,please input row again:"))
        b = int(input("please input column:"))
        if b > 3 or a < 1:
            b = int(input("Out of range,please input column again:"))
        while borad[a-1][b-1] != 0:
            a = int(input("The position has been occupied,please input row again:"))
            b = int(input("The position has been occupied,please input column again:"))
        borad[a-1][b-1] = 1
    else:
        print("now player is 2")
        a = int(input("please input row:"))
        while a > 3 or a < 1:
            a = int(input("Out of range,please input row again:"))
        b = int(input("please input column:"))
        while b > 3 or a < 1:
            b = int(input("Out of range,please input column again:"))
        while borad[a - 1][b - 1] != 0:
            a = int(input("The position has been occupied,please input row again:"))
            b = int(input("The position has been occupied,please input column again:"))
        borad[a - 1][b - 1] = 2

def condition(borad):
    for i in range(3):
        if borad[i][0] == borad[i][1] == borad[i][2] and borad[i][0]!= 0:
            print("winer is {}".format(borad[i][0]))
            return False
    for i in range(3):
        if borad[0][i] == borad[1][i] == borad[2][i] and borad[0][i]!= 0:
            print("winer is {}".format(borad[0][i]))
            return False
    if (borad[0][0] == borad[1][1] == borad[2][2] and borad[1][1] != 0) or (borad[2][0] == borad[1][1] == borad[0][2] and borad[1][1] != 0):
        print("winer is {}".format(borad[1][1]))
        return False
    return True

def mainprogram():
    game_mune()
    display_borad()
    count = 0
    while condition(borad) or full_borad(borad) :
        try:
            player(count)
        except:
            print("Program error, please re-enter according to the rules!")
            player(count)
        count += 1
        display_borad()

if __name__ == '__main__':
    borad = [[0,0,0] for i in range(3)]
    mainprogram()

程序分析

核心函数就是game_mune()，display_borad()，full_borad(borad)，player(count)，condition(borad)，mainprogram()。

game_mune()

提醒玩家游戏开始

display_borad()

通过”|”和”–”构造类似棋盘的形状，行和列就是列表的嵌套结构的索引值-1

full_borad(borad)

通过判断0是否存在于board的棋盘中，判断棋盘是否填充满

condition(borad)

判断获胜条件，分别从横向、纵向和对角线三个方向进行判断，如果有一个满足条件则返回False，终止在mainprogram()的循坏条件，如果三个条件都不满足，则继续返回True，mainprogram()中继续循环

player(count)

通过判断count的奇偶性来判断玩家的变换，这个部分最关键的玩家输入的合法性，首先对输入的范围进行判断，然后在整个过程中如果出现输入值不为int函数的情况，将在mainprogram()中捕获异常，根据输入的行数和列数对borad进行修改

mainprogram()

mainprogram()当获胜条件和平局条件不符合时,游戏将一直执行

局限性

代码的健壮性并不强，比如在这种写法中，输入行和列的时候，如果输入不是int，只能执行一次try函数，也就意味着只有一次修改输入错误行、列的机会

遇到的问题

产生board的时候采用了

[[0,0,0]]*3

发现修改索引值[0][0],borad变成了

[[1,0,0],[1,0,0],[1,0,0]]

在复制*3的过程中，id地址一样，所以导致了这种错误
所以可以修改为一下两种写法

[[0,0,0] for i in range(3)]
[[0,0,0],[0,0,0],[0,0,0]]
