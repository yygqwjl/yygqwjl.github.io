eetcode 1493. 删掉一个元素以后全为 1 的最长子数组 Medium

题目

给你一个二进制数组 nums ，你需要从中删掉一个元素。
请你在删掉元素的结果数组中，返回最长的且只包含 1 的非空子数组的长度。
如果不存在这样的子数组，请返回 0 。

code

class Solution:
    def longestSubarray(self, nums) -> int:
        if 0 not in nums:
            return len(nums) - 1
        loca = []
        for i in range(len(nums)):
            if nums[i] == 0:
                loca.append(i+1)
        if nums[0] != 0:
            loca.insert(0,0)
        if nums[-1] != 0:
            loca.append(len(nums)+1)
        result = []
        for i in range(1,len(loca)):
            result.append(loca[i]-loca[i-1]-1)
        nn = []
        for i in range(1,len(result)):
            nn.append(result[i]+result[i-1])
        return max(nn)

题解

统计出所有的0的位置，然后两个0的位置之差-1为一段只包含1的子数组，分别对两个相邻的子数组的长度相加，找出最长的长度。

需要注意的是类似于种树问题，判断两个边界。

leetcode1498. 满足条件的子序列数目 Medium

题目

给你一个整数数组 nums 和一个整数 target 。

请你统计并返回 nums 中能满足其最小元素与最大元素的 和 小于或等于 target 的 非空 子序列的数目。

由于答案可能很大，请将结果对 10^9 + 7 取余后返回。

code

暴力超时

class Solution:
    def numSubseq(self, nums, target: int) -> int:
        result = []
        nums.sort()
        for i in range(len(nums)):
            for j in range(i,len(nums)):
                if nums[i] + nums[j] <= target:
                        result.append([i,j])
        adic = {}
        for i in result:
            if i[0] in adic:
                if i[1] > adic[i[0]]:
                    adic[i[0]] = i[1]
            else:
                adic[i[0]] = i[1]
        k = 0
        for i in adic:
            k += 2**(adic[i]-i)
        return k%(10**9+7)

优化

class Solution:
    def numSubseq(self, nums: List[int], target: int) -> int:
        nums.sort()
        ret = 0
        n = len(nums)
        j = n - 1
        S = 10 ** 9 + 7
        for i in range(n):
            while j >= i and nums[j] + nums[i] > target:
                j -= 1
            if j < i:
                break
            ret = (ret + 2 ** (j - i)) % S
        return ret

题解

首先sort，然后找出每个i到另一个比i大的数的最大间隔，结果就是2**每个最大间隔之和。

这里用我写的暴力法更好理解，就是高中学习过的排列组合。

nums = [3,3,6,8], target = 10
result = [[0, 0], [0, 1], [0, 2], [1, 1], [1, 2]]
adic = {0: 2, 1: 2}
#这里的0，1，2都是在nums中的序号，并不是nums中的数

最后的k就是2**(adic[i]-i)之和

排列组合思想：
对于每个数，找到最多满足条件的数的个数，然后2**n

暴力是在找最多的数过程中，进行了两次On所以超时，建立字典这个操作是到了最后时刻实在想不出来的比较蠢的方法，看了优化才恍然大悟。

leetcode 1503. 所有蚂蚁掉下来前的最后一刻 Medium

题目

有一块木板，长度为 n 个 单位 。一些蚂蚁在木板上移动，每只蚂蚁都以 每秒一个单位 的速度移动。其中，一部分蚂蚁向 左 移动，其他蚂蚁向 右 移动。

当两只向 不同 方向移动的蚂蚁在某个点相遇时，它们会同时改变移动方向并继续移动。假设更改方向不会花费任何额外时间。

而当蚂蚁在某一时刻 t 到达木板的一端时，它立即从木板上掉下来。

给你一个整数 n 和两个整数数组 left 以及 right 。两个数组分别标识向左或者向右移动的蚂蚁在 t = 0 时的位置。请你返回最后一只蚂蚁从木板上掉下来的时刻。

code

class Solution:
    def getLastMoment(self, n: int, left: List[int], right: List[int]) -> int:
        return max(left+[n-i for i in right])

题解

刚看这道题直接懵了，想到了原来看到过的弹球模拟，但是心想太复杂了，后来看到群友说是到益智类的题目，所以通过看案例看出了答案。
、

当然也付出了WA了五次的代价。

这个题主要是理解碰撞。
(以下参考为力扣本题的官方题解)
由于改变移动方向不花费额外时间，而且改变移动方向后的移动速度不变，因此，两只相遇的蚂蚁同时改变移动方向之后的情形等价于两只蚂蚁都不改变移动方向，继续按照原来的方向和速度移动，这样问题就简化成根据每只蚂蚁的初始位置和移动方向得到最后一只蚂蚁到达木板边界的时刻。

leetcode 1509. 三次操作后最大值与最小值的最小差 Medium

题目

给你一个数组 nums ，每次操作你可以选择 nums 中的任意一个元素并将它改成任意值。

请你返回三次操作后， nums 中最大值与最小值的差的最小值。

code

class Solution:
    def minDifference(self, nums: List[int]) -> int:
        if len(nums) <= 4: return 0
        nums.sort()
        return min(nums[-1] - nums[3], nums[-2] - nums[2], nums[-3] - nums[1], nums[-4] - nums[0])

题解

此题类似于上面的蚂蚁的题，都是属于脑筋急转弯的范围，三次操作能对结果造成影响的话，是只改变最大值，最小值，才会对nums中最大值与最小值的差有影响，所以考虑修改的几种情况，维护最大值和最小值就可以了

nums[-1] - nums[3] 去掉3个最小的

nums[-2] - nums[2] 去掉1个最大的，去掉2个最小的

nums[-3] - nums[1] 去掉2个最大的，去掉1个最小的

nums[-4] - nums[0] 去掉三个最大的

然后在这三个结果中找出最小的一个就可以了
