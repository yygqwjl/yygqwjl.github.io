format方法格式化

部分内容引用自 https://blog.csdn.net/jpch89/article/details/84099277

题外话

因为最早学习python的时候只掌握了 % 方法对字符进行格式化，现在系统学习学习一下format方法，目前我发现的区别是在补充空格format可以自定义字符进行填充

标准格式

[[填充]对齐方式][正负号][#][0][宽度][分组选项][.精度][类型码]

填充与对齐方式

填充

只能是一个字符
不指定默认用空格填充
如果指定填充字符，则必须要同时指定对齐方式

对齐方式

<：左对齐

>：右对齐

^：居中

正负号

正负号选项仅对数字类型生效
取值有三种：

  • 正数前面添加正号，负数前面添加负号
  • 仅在负数前面添加负号（默认行为）

    #号

    给u二进制数加上 0b 前缀
    给八进制数加上 0o 前缀
    给十六进制数加上 0x 前缀

    最小宽度

    如果不指定，最小字段宽度由内容决定，与内容相等
    如果最小宽度前面有一个前导 0，意味着用 0 填充
    这等价于指定了 0= 的填充和对齐方式

    分组选项

    逗号,

    使用逗号对数字以千为单位进行分隔 n 类型的数字可以使用本地化的分隔符。

    下划线_

    使用下划线对浮点数和 d 类型的整数以千为单位进行分隔。对于 b、o、x 和 X 类型，每四位插入一个下划线，其他类型都会报错。

    精度

    精度指定了小数点后面要展示多少位小数
    对于非数字类型，精度指定了最大字段宽度
    整数类型不能指定精度

    类型码

    字符串类型

    s 字符串类型。这是字符串的默认类型，可以省略。
    None 不指定类型。同 s 类型。

    整数类型

    b(bin) 二进制
    c unicode
    d(int) 十进制
    o(oct) 八进制
    x(hex) 十六进制

    浮点数类型

    f 定点记法，默认精度为 6
