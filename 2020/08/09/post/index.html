5483. 整理字符串

题目

给你一个由大小写英文字母组成的字符串 s 。

一个整理好的字符串中，两个相邻字符 s[i] 和 s[i + 1] 不会同时满足下述条件：

0 <= i <= s.length - 2
s[i] 是小写字符，但 s[i + 1] 是相同的大写字符；反之亦然。

请你将字符串整理好，每次你都可以从字符串中选出满足上述条件的 两个相邻字符并删除，直到字符串整理好为止。

请返回整理好的字符串 。题目保证在给出的约束条件下，测试样例对应的答案是唯一的。

注意：空字符串也属于整理好的字符串，尽管其中没有任何字符。

code

class Solution:
    def makeGood(self, s: str) -> str:
        def main(s):
            for i in range(len(s)-1):
                if (s[i].islower() and s[i+1].isupper() and s[i].upper() == s[i+1]) or (s[i+1].islower() and s[i].isupper() and  s[i].lower() == s[i+1]):
                    return (True,i)
            return [False,float(inf)]
        while main(s)[0] == True:
            m = main(s)[1]
            s = s[:m] + s[m+2:]
        return s

题解

这题是我遇到过的最难的周赛T1，面对这种直到什么什么为止，直接用while，并且本题数据量是

1 <= s.length <= 100

所以直接暴力，O(n^2)，while条件中存在两个相邻字符满足条件，返回True和字符位置，然后把这两个字符串删除。

5469. K 次操作转变字符串

题目

给你两个字符串 s 和 t ，你的目标是在 k 次操作以内把字符串 s 转变成 t 。

在第 i 次操作时（1 <= i <= k），你可以选择进行如下操作：

选择字符串 s 中满足 1 <= j <= s.length 且之前未被选过的任意下标 j （下标从 1 开始），并将此位置的字符切换 i 次。
不进行任何操作。
切换 1 次字符的意思是用字母表中该字母的下一个字母替换它（字母表环状接起来，所以 ‘z’ 切换后会变成 ‘a’）。

请记住任意一个下标 j 最多只能被操作 1 次。

如果在不超过 k 次操作内可以把字符串 s 转变成 t ，那么请你返回 true ，否则请你返回 false 。

code

class Solution:
    def canConvertString(self, s: str, t: str, k: int) -> bool:
        if len(s) != len(t):
            return False
        zimu = "ABCDEFGHIJKLMNOPQRSTUVWXYZ"
        mm = {}
        for i in range(1,27):
            mm[i] = k//26
        for i in range(1,k%26+1):
            mm[i] += 1
        adic = {}
        for i,j in zip(zimu.lower(),range(1,27)):
            adic[i] = j
        for i,j in zip(s,t):
            if i != j:
                m = adic[j]-adic[i]
                if m < 0:
                    m += 26
                if mm[m] <= 0:
                    return False
                else:
                    mm[m] -= 1
        return True

题解

首先两个字符串的长度不同，直接返回False。

t字符与s字符不同，肯定能在26步操作内变为一致，(一次切换是用字母表中该字母的下一个字母替换它)，首先建立字母表的字典，对应[1,26]，t字符的key值减去s字符的key值如果为正值，则该值就是操作需要的次数，如果小于0，需要+26，则为操作的次数。

这里也可以不用字典，用ord方法也可以。

k次操作每次取模26即为能改变的次数。

mm字典中统计k内每种需要改变的次数的次数。

如果这种次数为0，并且还需要这种次数的操作，则返回False。

#5470. 平衡括号字符串的最少插入次数

题目

给你一个括号字符串 s ，它只包含字符 ‘(‘ 和 ‘)’ 。一个括号字符串被称为平衡的当它满足：

任何左括号 ‘(‘ 必须对应两个连续的右括号 ‘))’ 。
左括号 ‘(‘ 必须在对应的连续两个右括号 ‘))’ 之前。
比方说 “())”， “())(())))” 和 “(())())))” 都是平衡的， “)()”， “()))” 和 “(()))” 都是不平衡的。

你可以在任意位置插入字符 ‘(‘ 和 ‘)’ 使字符串平衡。

请你返回让 s 平衡的最少插入次数。

code

class Solution:
    def minInsertions(self, s: str) -> int:
        stack = []
        ans = 0
        i = 0
        while i<len(s):
            if s[i] == '(': #左括号直接入栈
                stack.append('(')
            else:
                if i == len(s)-1:#右括号，情况一，为最后一个符号
                    if stack: #如果栈不为空，说明有左括号，补一个右括号
                        ans += 1
                        stack.pop()
                    else: #空栈那么需要补一个左括号和一个右括号
                        ans += 2
                else: #情况二，不是最后一个字符
                    if stack: #如果栈不为空，说明有左括号，看后面有没有右括号
                        if s[i+1] == ')': #完美匹配，弹出
                            stack.pop()
                            i += 1
                        else: #为左括号，说明需要补充
                            stack.pop()
                            ans += 1
                    else:
                        if s[i+1] == ')': #前面补充一个左括号
                            i += 1
                            ans += 1
                        else: #为左括号，说明需要补充一个左一个右
                            ans += 2
            i += 1
        #最后剩下栈里的左括号
        return ans + len(stack)*2

题解

这道题看到括号处理就知道要使用栈思想，但是在竞赛过程处理不够完美，所以只过了案例，没有AC。这份代码是赛后抄的。注释也很完美，主要是要把所有情况考虑清楚。

左括号入栈。

右括号如果为最后一个符号，如果栈不为空，说明有左括号，补一个右括号，pop掉一个左括号，ans+=1，如果栈为空，需要补一个左括号和右括号。

如果右括号不为最后一个字符，如果栈不为空，匹配后面有没有另一个右括号，如果有的话，成功进行了匹配，只需要pop一个左括号，并且索引值+2。如果下一个为左括号，pop掉一个左括号，ans+=1

如果栈为空，并且下一个字符为右括号，则只需要补充一个左括号，ans+=1。如果下一个左括号，则需要补充两个，一个左括号，一个右括号。

这道题主要在于一个仔细。
